## v1.0.0 — Initial release

First public release of the Notifications app.

### Highlights
- Firebase Auth (email/password) with online status tracking
- Push alerts with custom sound and persistent dialog
- Reports / history view
- Self-update from GitHub Releases (`UpdateChecker`)
- Signed release build

### Install
Download the attached `notifications-v1.0.0.apk` and install it on Android 7.0+ (minSdk 24).
You may need to allow **"Install from unknown sources"** when prompted.

- `versionCode`: 2
